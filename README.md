Crowd Monitoring System using AI

Project Description
The Crowd Monitoring System is an AI-powered application developed as part of the Infosys Springboard Project. This system uses computer vision and deep learning
(YOLOv8) to detect, monitor, and analyze crowd density in real-time from video feeds.
The project is designed to improve public safety and crowd management by automating the process of detecting and counting people in specific zones. It provides
data analytics and reporting features to understand crowd patterns over time.
